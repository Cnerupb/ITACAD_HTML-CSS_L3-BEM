# HTML-CSS Уровень 2 - вёрстка макета

## Зависимости

- Node.js
- Браузер

## Настройка окружения

1. Переместите файлы проекта в пустую папку

2. Выполните в папке проекта в терминале команду:
```
npm install
```

## Запуск

3. Запустить проект можно в терминале командой:
```
npm run dev
```
- В терминале запустится сервер и отобразится адресс для просмотра в браузере.
